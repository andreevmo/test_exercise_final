Проверить выполнение запроса с urls корректной формы

1. Выполнить запрос POST по адресу "https://cleanuri.com" на сервис "/api/v1/shorten" с url из файла correct_url.csv
2. Проверить успешность(ожидаемый код 200)
3. Проверить, что в ответе пришел укороченный url, все результаты начинаются с "https://cleanuri.com/"

Проверить выполнение запроса с urls некорректной формы

1. Выполнить запрос POST по адресу "https://cleanuri.com" на сервис "/api/v1/shorten" с url из файла incorrect_url.csv
2. Проверить успешность(ожидаемый код )
3. Проверить, что в ответе пришел body с описанием ошибки
